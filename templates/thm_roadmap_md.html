The Ultimate Guide for Beginners

1. Beginner Level Rooms

Perfect for those new to cybersecurity, these rooms help users understand foundational concepts in a hands-on environment.

  • Introduction to Cyber Security: Beginner-friendly intro covering key principles.
  • Linux Fundamentals (Part 1, 2, 3): Essential commands and functions of Linux.
  • Intro to Defensive Security: Basic defensive measures and secure environment setup.
  • Phishing Emails: Identifying phishing attacks and malicious links.
  • Network Fundamentals: Basics of networking, IP addresses, protocols.

2. Intermediate Level Rooms

After understanding the basics, these rooms help build essential skills for ethical hacking and network security.

  • Pre-Security Path: Foundational layers before in-depth content.
  • Introduction to Web Hacking: SQL Injection, XSS, CSRF.
  • OWASP Top 10: Common web application vulnerabilities.
  • Hash Cracking: Password cracking with Hashcat and John the Ripper.
  • File Inclusion: Spotting and exploiting file inclusion vulnerabilities.

3. Advanced Level Rooms

For those ready to tackle complex vulnerabilities and comprehensive penetration testing.

  • Red Teaming Fundamentals: Emulating adversary tactics.
  • Pivoting: Lateral movement within networks.
  • Privilege Escalation: Exploiting vulnerabilities to escalate privileges (Linux/Windows).
  • Active Directory Basics: Attacking and securing AD infrastructures.
  • Binary Exploitation: Buffer overflows and reverse engineering.

4. Capture the Flag (CTF) Challenges

  • Pickle Rick: Rick and Morty themed web exploitation.
  • RootMe: Gain initial access and escalate privileges.
  • Blue: Exploiting EternalBlue.
  • Mr. Robot: Storyline-driven CTF based on the show.
  • Ice: Enumeration and exploitation focus.

5. Topic-Specific Rooms

  • Burp Suite Basics: Essential tool for web testing.
  • Metasploit Basics: Using the Metasploit Framework.
  • Malware Analysis: Intro to analyzing malware behavior.
  • Wireshark Basics: Packet analysis basics.
  • Steganography CTFs: Hiding messages within files.
